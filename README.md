# 宝石連盟オンライン 💎

宝石をテーマにしたクリッカーゲームです。4つの宝石グループ（ルビー、サファイア、エメラルド、トパーズ）に分かれて、クリックで資産を増やし、トレードで通貨を交換できます。

## 🎮 ゲームの特徴

- **4つの宝石グループ**: ルビー（赤）、サファイア（青）、エメラルド（緑）、トパーズ（黄）
- **クリックで資産獲得**: 画面をクリックしてグループの総資産を増やします
- **通貨トレード**: Common通貨と各グループ通貨を交換できます
- **動的な為替レート**: トレードによって為替レートが変動します
- **状態の自動保存**: ブラウザのlocalStorageにゲーム状態を自動保存

## 🎨 デザイン

- **ゲーム風UI**: ピクセルアート風フォントとネオンエフェクト
- **宝石のアニメーション**: 各グループに応じた宝石の色と輝き
- **モーダルウィンドウ**: トレード、グループ総資産、為替レートをモーダルで表示
- **レスポンシブ対応**: モバイルデバイスでも快適にプレイ可能

## 🚀 使い方

### 1. グループ選択
ゲーム開始時に4つの宝石グループから1つを選択します。

### 2. クリックで資産獲得
- 画面のどこでもクリック（またはタップ）して資産を獲得
- クリックすると所属グループの総資産が+1されます
- 0.2秒のクールダウンがあります

### 3. トレード
- ヘッダーの「トレード」ボタンをクリック
- From通貨、To通貨、数量を選択して送信
- トレードによって為替レートが変動します

### 4. 情報確認
- **グループ総資産**: 各グループの総資産を確認
- **為替レート**: 各グループ通貨とCommon通貨のレートを確認
- **エメラルド情報**: 左カラムにエメラルドの総資産とレートが常時表示

## 🎯 ゲームルール

### 資産獲得
- クリック1回でCommon通貨+1、所属グループの総資産+1

### トレード
- Common通貨と各グループ通貨を交換可能
- グループ通貨同士の交換も可能（Common経由）
- トレードによって為替レートが変動：
  - グループ通貨を売ってCommonを買う → グループ通貨安・Common高（レート+1）
  - Commonを売ってグループ通貨を買う → グループ通貨高・Common低（レート-1）
- レートは1〜4の範囲で変動

### 為替レート
- 初期レート: 各グループ通貨 1 : Common 1
- レート変動: トレードによって動的に変化

## 💾 データ保存

ゲームの状態はブラウザのlocalStorageに自動保存されます：
- 選択したグループ
- 獲得した通貨（Common通貨、グループ通貨）
- 各グループの総資産
- 為替レート
- クリック数

ブラウザを更新しても状態が保持されます。

## 🔧 カスタマイズ

### グループ名の変更
`main.js`の`groupNames`オブジェクトを編集してください。

### 色の変更
`main.js`の`groupColors`オブジェクトと`style.css`の各グループの色定義を編集してください。

### クールダウン時間の変更
`main.js`の`CLICK_COOLDOWN`定数を変更してください（デフォルト: 200ms）。
